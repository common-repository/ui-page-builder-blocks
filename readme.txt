Interactive UI blocks for building the pages of your site. Includes image sliders and image viewers.

== Description ==

[UI Page Builder Blocks](http://www.wpexpertdeveloper.com/ui-page-builder-blocks/) is an awesome set of UI components for building the pages and functionality of your site. This plugin contains image sliders and image viewers.

> **Features**
>
> * Simple and Flexible Image Slider
> * Unlimited Image Galleries
> * Unlimited Logo/ Thumbnail Sliders
> * Unlimited Vertical Image Sliders
> * Unlimited Tab Sliders
> * Create awesome Image Viewer(Gallery) with Modal view or Inline view.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ui-page-builder-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Follow the <a href="http://www.wpexpertdeveloper.com/ui-page-builder-blocks/">Documentation and Usage</a>.
